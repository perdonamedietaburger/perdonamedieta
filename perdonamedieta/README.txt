
PERDONAME DIETA BURGER - Proyecto listo para Vercel
Dominio configurado: perdonamedieta.com.ar
Moneda: ARS
Costo delivery: $1500

Instrucciones básicas y variables están en .env.example

Para desarrollar local:
1) npm install
2) copiar .env.example a .env.local y completar credenciales (MERCADOPAGO, SMTP o SENDGRID)
3) npm run dev

Subir a Vercel:
- Subir este ZIP en tu cuenta de Vercel o conectar un repo.
- Configurar las Environment Variables en el panel de Vercel con las mismas keys del .env.example
- Deploy.

Si necesitás que yo configure las credenciales o pruebe el webhook con ngrok, avisame.
