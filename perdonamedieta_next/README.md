
Perdoname Dieta - Next.js App Router project
- Next.js 14 (App Router)
- Tailwind config included
- Cart (localStorage), Checkout (Mercado Pago), Webhook (email notification)
- data/products.json editable

Local:
npm install
copy .env.example to .env.local and fill keys
npm run dev

Deploy to Vercel: push to GitHub and import (Next.js detected automatically)
